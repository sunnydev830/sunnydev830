### 🚀 About Me

I’m a **Back-End Developer & AI Engineer** with a strong focus on:

- 🚀 Scalable microservices using FastAPI, Django, and Express.js  
- 📊 Real-time data processing and Redis-based queues  
- 🧠 AI integration with LLMs, vector DBs, and NLP pipelines  
- 📡 API design for seamless frontend/backend communication

I specialize in building high-performance systems that are secure, efficient, and AI-ready.
